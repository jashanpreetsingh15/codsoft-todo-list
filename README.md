# codsoft-todo-listTo-Do List Application

Description

A simple Python command-line To-Do List application that helps users manage tasks. Users can add, view, update, complete, and delete tasks.

Features

- Add Task
- View Tasks
- Mark Task as Done
- Update Task
- Delete Task
- Exit Program

Requirements

- Python 3.x

Run

python todo.py
